# JW-Apps

## Logging Application (Excel + VBA)

This repository contains the development of a geotechnical logging tool designed for:
- Test pit logging
- Borehole logging (future extension)
- dotPLOT-compatible export

---

## Structure

```
JW-Apps/
│
├── vba/
│   └── modTPLogger.bas        # Main VBA logic
│
├── excel/
│   └── (future template files)
│
└── README.md
```

---

## Current Features

- Create new test pit sheets from template
- Auto-build index of all pits
- Extract sample list from notes
- Structured logging workflow
